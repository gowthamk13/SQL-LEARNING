# learning-sql

Welcome! This repository is designed for learning and practicing SQL using example scripts and queries. The contents include table creation, initial data inserts, and a wide variety of query examples showcasing important SQL concepts. 

## File Structure

- `01-create-tables.sql`: Create example tables for students, courses, and enrollments.
- `02-insert-data.sql`: Insert starter data.
- `03-basic-queries.sql`: Simple select and join examples.
- `04-extended-queries.sql`: 15 unique queries covering aggregation, join, subquery, delete, update, etc.
- `05-more-unique-queries.sql`: 15 additional queries demonstrating filtering, string functions, DISTINCT, CASE, complex joins, and more.
